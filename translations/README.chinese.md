# My Website

这是我的个人网站，用于展示我的作品和分享我的想法。

## 技术栈

- 前端：HTML、CSS、JavaScript、React
- 后端：Node.js、Express、MongoDB

## 功能

- 展示我的作品和项目
- 展示我的博客文章
- 提供联系方式

## 安装

1. 克隆此仓库：`git clone https://github.com/username/my-website.git`
2. 进入仓库目录：`cd my-website`
3. 安装依赖：`npm install`
4. 启动应用：`npm start`
