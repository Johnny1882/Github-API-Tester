## Listes

Il y a deux types de listes en Markdown : les listes ordonnées et les listes non ordonnées.

### Listes non ordonnées

- Élément 1
- Élément 2
- Élément 3

### Listes ordonnées

1. Élément 1
2. Élément 2
3. Élément 3
