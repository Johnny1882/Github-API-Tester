# 你好世界!!!

欢迎来到我的应用程序

## 介绍

这个应用程序是用来展示如何使用 `Flask` 框架创建简单的Web应用程序。

## 安装

使用以下命令安装所有必需的软件包：

```bash
pip install -r requirements.txt
```

## 运行应用程序

在终端中运行以下命令：

```bash
export FLASK_APP=app.py
flask run
```

现在，您可以通过访问 `http://127.0.0.1:5000` 在浏览器中查看应用程序。
