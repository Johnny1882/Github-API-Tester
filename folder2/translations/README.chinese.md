## Usage

只需打开此文件并查看它的源代码，就可以了解如何使用Markdown。

### Headers

标题

使用“#”符号表示标题，数量越多，标题级别越高。

### Emphasis

强调

使用“*”或“_”符号来强调文本。

### Lists

列表

使用“-”或“*”符号来创建无序列表，使用数字和“.”来创建有序列表。

### Links

使用方括号“[]”和圆括号“()”来创建链接。

### Code

使用反引号“`”来标记代码块。

### Images

使用“!”和链接语法来添加图片。
