## 语法

Markdown 语法很简单，但非常实用。以下是一些基本语法：

- 标题：使用 `#` 符号表示标题级别，共有 6 级。
- 列表：使用 `-` 或 `*` 符号表示无序列表，数字加 `.` 表示有序列表。
- 强调：使用 `*` 或 `_` 符号表示斜体，使用 `**` 或 `__` 符号表示加粗。
- 代码：使用 \` 符号表示行内代码，使用 ``` 符号表示代码块。
