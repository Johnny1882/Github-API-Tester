# 你好世界!!!

## 介绍

这是一个简单的“Hello World”程序，用于展示如何在控制台上输出文本。

## 使用方法

1. 打开终端或命令提示符。
2. 导航到程序所在的文件夹。
3. 运行 `python hello_world.py` 命令。
4. 在控制台上会输出 `Hello, world!`。

## 代码

```python
# 定义一个字符串变量，存储要输出的文本
message = "Hello, world!"

# 在控制台上输出文本
print(message)
```

## 结论

这个程序很简单，但是它展示了如何在控制台上输出文本。这是编程的基础，对于初学者来说是一个很好的起点。
